Tudo oque você precisa saber de Carteiras Bitcoin

O que é uma Carteira Bitcoin?

Uma carteira Bitcoin é um software ou hardware que permite armazenar, enviar e receber Bitcoins. Ela contém as chaves privadas necessárias para assinar transações e as chaves públicas que permitem receber Bitcoin. As carteiras Bitcoin podem ser de vários tipos, incluindo carteiras de software, carteiras de hardware, carteiras de papel, e carteiras online (custodiais e não-custodiais).

O que é uma Seed?

Uma seed (ou seed phrase) é uma sequência de 12, 18 ou 24 palavras geradas pela carteira de Bitcoin durante a configuração inicial. Essas palavras representam um backup da carteira e podem ser usadas para recuperar os fundos em caso de perda ou dano do dispositivo. É crucial manter a seed em um lugar seguro e nunca compartilhá-la.

O que é uma Passphrase?

Uma passphrase é uma senha adicional que pode ser combinada com a seed phrase para fornecer uma camada extra de segurança. Mesmo que alguém obtenha a seed phrase, sem a passphrase eles não poderão acessar os fundos. É importante não esquecer a passphrase, pois ela é necessária para a recuperação da carteira.

O que é uma Chave-Mestra (Master Key)?

Uma chave-mestra é uma chave criptográfica derivada da seed que pode ser usada para gerar todas as chaves privadas e públicas subsequentes na carteira. Isso permite que uma única seed ou chave-mestra gerencie múltiplos endereços Bitcoin e transações.

O que é uma Chave Privada?

Uma chave privada é um código secreto que permite ao proprietário gastar os Bitcoins associados a uma chave pública correspondente. É essencial manter a chave privada segura, pois qualquer pessoa com acesso a ela pode gastar os Bitcoins da carteira.

O que é uma Chave Pública?

Uma chave pública é derivada da chave privada e pode ser compartilhada publicamente. Ela é usada para gerar endereços Bitcoin e permite que outros enviem Bitcoins para a carteira associada. A chave pública não permite gastar os Bitcoins, apenas recebê-los.

O que é uma Extended Public Key (xpub)?

Uma extended public key (xpub) é uma chave pública estendida que permite gerar múltiplos endereços Bitcoin sem revelar a chave privada. Isso é útil para carteiras que desejam gerar novos endereços para cada transação recebida sem comprometer a segurança.

O que é um Endereço?

Um endereço Bitcoin é uma string alfanumérica derivada de uma chave pública. Ele funciona como um destino para onde os Bitcoins podem ser enviados. Cada endereço é único e pode ser usado para receber Bitcoin.

O que é uma Carteira Multisig?

Uma carteira multisig (multi-assinatura) requer múltiplas assinaturas de diferentes chaves privadas para autorizar uma transação. Isso aumenta a segurança, pois a perda de uma única chave privada não compromete a segurança dos fundos. É frequentemente usada para transações corporativas ou compartilhadas.

O que é uma Carteira Watch-Only?

Uma carteira watch-only permite visualizar o saldo e as transações de uma carteira Bitcoin sem ter acesso à chave privada. Isso é útil para monitoramento e auditoria, pois não permite o envio de Bitcoins.

O que é uma Carteira Custodial?

Uma carteira custodial é uma carteira onde um terceiro (como uma exchange) possui e gerencia as chaves privadas em nome do usuário. Embora seja conveniente, isso significa que o usuário não tem controle total sobre seus fundos.

O que é uma Carteira Não-Custodial?

Uma carteira não-custodial é uma carteira onde o usuário possui e controla suas próprias chaves privadas. Isso oferece maior segurança e controle, pois apenas o usuário tem acesso aos seus fundos.

O que é uma Hardware Wallet (Hardwallet)?

Uma hardware wallet é um dispositivo físico que armazena chaves privadas offline, protegendo-as de malware e hackers. Exemplos incluem Trezor e Ledger. É considerada uma das formas mais seguras de armazenar Bitcoin.

O que é uma Cold Wallet (Carteira Fria)?

Uma cold wallet é uma carteira Bitcoin que não está conectada à internet, tornando-a imune a ataques online. Isso inclui hardware wallets e paper wallets. Cold wallets são usadas para armazenar grandes quantidades de Bitcoin de forma segura.

O que é uma Hot Wallet (Carteira Quente)?

Uma hot wallet é uma carteira Bitcoin conectada à internet, facilitando o acesso e a realização de transações rápidas. No entanto, devido à sua conexão constante, é mais vulnerável a ataques cibernéticos. Exemplos incluem carteiras de software em smartphones e desktops.

O que é uma Paper Wallet (Carteira de Papel)?

Uma paper wallet é um pedaço de papel onde as chaves públicas e privadas de uma carteira Bitcoin são impressas. Por ser completamente offline, é considerada uma forma segura de armazenamento frio. No entanto, é vulnerável a danos físicos e perda, então deve ser armazenada em um local seguro.
